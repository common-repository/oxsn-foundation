This plugin adds fundamental styles/shortcodes/quicktags/and more!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/oxsn-foundation` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->OXSN Foundation screen to configure the plugin
